# Restaurant Trust Analytics & Fake Review Detection System

## Overview

The Restaurant Trust Analytics & Fake Review Detection System is a data analytics project designed to analyze restaurant ratings, customer votes, and engagement patterns to evaluate restaurant trustworthiness.

The project uses Python for data cleaning, preprocessing, exploratory data analysis (EDA), and risk analysis, while Power BI is used to create interactive dashboards and business intelligence reports.

This project demonstrates practical skills in Data Analytics, Data Visualization, Business Intelligence, and Insight Generation.

---

## Problem Statement

Online restaurant platforms rely heavily on customer ratings and reviews. However, ratings alone may not always reflect the true quality of a restaurant.

The goal of this project is to analyze restaurant ratings and customer engagement data to identify unusual patterns and assess the reliability of restaurant ratings.

---

## Objectives

* Clean and preprocess restaurant data.
* Perform Exploratory Data Analysis (EDA).
* Analyze restaurant ratings and voting patterns.
* Create a trust analytics framework using risk scores.
* Develop interactive Power BI dashboards.
* Generate business insights and recommendations.

## Key Findings

* Ratings alone are not sufficient indicators of restaurant trustworthiness.
